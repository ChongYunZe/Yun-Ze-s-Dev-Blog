Week 5 - Introduction To JavaScript

What I learnt:

  • Syntax
  • Control Structures
  • Variables
    • Declaring & Assigning Variables
    • var - Allows Variable To Be Changed Later
    • const - Variable Cannot Be Changed
    • let variable = ...
  • Types Of Data
    • Strings
      • .toUpperCase()
      • .length
      • "${variable}" - Formatting
    • Numbers
      • Infinity
      • -Infinity
      • NaN
      • Math
        • .round()
        • .random()
    • Booleans
      • true/false - Small Letters
      • ===/!== - Strict Equality Function
      • ==/!= - Not Strict Equality Function
    • Objects
      • Use {}
      • {"label":"value:"}
      • Data Modeling
      • ["label"]/.label
      • delete - Remove Object
      • Making Copies Of An Object
        • variable = Object.assign({}, variable)
        • variable = {...variable}
        • JSON.parse(JSON.stringify(variable))
    • Arrays
      • Use []
      • Can Store Any Type In It
      • .length
      • Array[index] = '...'
      • Add To End Of Array
        • array.push("...")
        • array[array.length] = '...'
      • array.pop() - Remove Item At The End
      • array.splice(index, number items to remove) - Remove Item At Specific Index
    • Readability
      • Whitespace
      • Comments
        • // - Line Comments
        • /* */ - Block Comments
    • Regular Expression
      • variable = /this/
        • variable.test
        • /this/i (Case Insensitive)
        • /^this/i (Check Beginning Of The String)
        • /this$/i (Checking End Of The String)
        • /ever.$/i (Checking End Of String) - The "." Checks For Anything
        • /ever\.$/i (Checking For End Of String) - Checks For Literal "."
  • Arithmetic Operators
    • +, -, *, /, % (Gives Remainder For Division), +=, ++, -=, --, *=
  • Logical Operators
    • && - And
    • || - Or
  • Conditionals
    • if / else if/ else
    • switch - case...: / default:
    • Terse ifs
      • One-liners
      • Ternaries
        • ? - If True
        • : - If False
  • Type Checking
    • typeof
    • hasOwnProperty()
    • isNaN()
  • Loops
    • For Loops
      • for (let i = 0; i => 10; i+=1) { ... }
      • for( var p in variable) { ... }
    • While Loops
      • while() {...}
      • do{} while(); - Run The Loop At Least Once
  • Set And Map
    • let mySet = new Set()
    • let myMap = new Map()
    • .add() - Add In Values
    • new Set(List) - Removes Duplicates Of Array/List
